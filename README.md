# Smart Personal Finance Analytic Dashboard

## Overview
This project is an interactive financial analytics dashboard developed using Power BI. 
It helps users analyze income, expenses, and financial trends through dynamic visual reports.

## Tools Used
- Power BI
- Microsoft Excel

## Features
- Monthly income vs expense analysis
- Category-wise expense breakdown
- KPI indicators
- Interactive filters and slicers
- Visual charts and reports

## Dataset
The dataset includes financial transaction records such as:
- Date
- Category
- Amount
- Income / Expense type

## How to Use
1. Download the .pbix file.
2. Open it in Power BI Desktop.
3. Refresh the dataset if required.
